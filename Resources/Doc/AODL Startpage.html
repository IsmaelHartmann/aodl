AODL

An OpenDocument Library

With this library you could simply generate OpenDocuments from your code. The OpenDocument standard is the default file format for the office products OpenOffice (since 2.0.0) and for Star Office.
The OpenDocument it self is zip archive which contains alle document specific contents and settings within a set of xml files. By using AODL you don't have to know all the knowlegde of the xml elements and the structure of the xml files. All this stuff is handled by the library. You simply have to know how your document should looks like. AODL is published under th LGPL Lesser Genraral Public License so feel free to use it within your commercial products. Wheter the library is total free, it will be nice if you place a info link to AODL within your project.

Visit the homepage for updates. AODL homepage

The following example show how simple it is to create a document with AODL:

              TextDocument td        = new TextDocument();
            td.New();

            Paragraph p            = new Paragraph(td, "P1");
            //Add some content
            p.TextContent.Add( new SimpleText((IContent)p, "Hello i'm simple text!"));
            //Add some formated content
            FormatedText ft = new FormatedText((IContent)p, "T1",
                                                " And i'm formated text!");
            ((TextProperties)ft.Style.Properties).Bold = "bold";
            p.TextContent.Add(ft);           

            //Add as document content
            td.Content.Add(p);

            //Generate and save
            td.SaveTo(td, "ParagraphTest.odt");

This code create an OpenDocument textdocument with :

Hello i'm simple text! And i'm formated text!
